The .zip file contains two folders, the first (Tutorials) contains all the tutorials seen a
lesson with the variations required as homework. There is also the additional tutorial with relation
and the integration project of the various exercises with report. They are present in this folder
also all the images and models used, you must specify the local path where they are located
the various images / models in your pc for each exercise to make them work.
In the second (Graphic Programming) there is a visual studio project with all the linked libraries already
and compiled in such a way as to be ready to run the various main and vertex / fragment shaders of each exercise,
you only need to change the path to your local PC path by specifying this path:
... / Main_Exercises / Graphic_Programming / Includes
and then
... / Tutorials_Main / Programming_Graphics / Libraries
